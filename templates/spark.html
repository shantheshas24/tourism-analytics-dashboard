⚡ Apache Spark Analysis

Big Data processing of 1.48M tourism reviews using Apache Spark — demonstrating scalable distributed analytics.

Why Spark? Apache Spark is the industry-standard framework for distributed Big Data processing. This dataset (1.48M rows) was analysed using PySpark to demonstrate scalability, SQL capabilities, and distributed aggregation techniques suitable for datasets 10x–100x larger.
